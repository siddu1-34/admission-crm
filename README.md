# Admission Management & CRM

## Overview

This is a backend system for managing college admissions with quota-based seat allocation.

## Features

* Master setup (Program, Quota)
* Applicant management
* Seat allocation with validation
* Admission confirmation
* Dashboard

## Tech Stack

* Node.js
* Express
* MongoDB Atlas

## Setup

```bash
npm install
npm run dev
```

## API Base URL

http://localhost:5000/api
